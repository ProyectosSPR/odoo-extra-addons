Módulo para factura electrónica en México (CFDI 4.0)

Agrega campos adiiconales para generar facturas electrónicas en México (CFDI 4.0)

Se pueden emitir facturas:

  • Factura timbrada.


  • Se anexa el archivo xml y pdf al enviar correo electrónico.


Configuración del Módulo

Configuraciones necesarias:

  1. Compañía
    2. En la pestaña “Información general” se deben llenar los siguientes campos:
    1. Código postal (Cualquier valor válido para pruebas)
    2. RFC (Para pruebas XIA190128J61)
    3. Habilitar el check box "CFDI MX" (disponible en versiones 17 y mayores)


    NOTA: En versiones Odoo 12 o inferiores el campo RFC se debe llenar en la pestaña CFDI

    En la pestaña CFDI se debe configurar la información:
    1. Razon social: Nombre que aparece en la constancia de situación fiscal o para pruebas Xenon Industrial Articles
    2. Serie factura: Prefijo que se quiera utilizar para la serie de la factura (opcional disponible en versiones 17 y menores)
    3. Serie complemento de pago: Prefijo que se quiera utilizar para la serie del complemento de pago(opcional disponible en versiones 17 y menores)
    4. Régimen fiscal: Régimen que aparece en la constancia de situación fiscal o para pruebas colocar “General de Ley Personas Morales”
    5. Proveedor de timbrado: Principal
    6. Modo prueba: Para modo producción no se debe habilitar para modo de pruebas se debe habilitar (palomear)
    7. Certificado Digital: Cargar los certificados (CSD) de la compañía y su contraseña. Para modo de pruebas seleccionar los archivos .cer y .key que están en la carpeta documentación. La contraseña para los certificados es 12345678a


    Si requiere validar si es correcto su certificado CSD, puede ingresar a la siguiente dirección y para validar:
    https://solucionfactible.com/sfic/capitulos/timbrado/CSD.jsp

  2. Zona horaria del usuario
    3. Entrar al perfil del usuario.

    Seleccionar la Zona horaria correspondiente al código postal ingresado.

  3. Impuestos
    4. Es necesario configurar todos los impuestos que se vaya a utilizar para timbrar las facturas electŕonicas.

    En la pestaña “CFDI” seleccionar el tipo de impuesto y el factor correspondiente al impuesto a configurar.

  4. Clientes
    5. Es necesario configurar 5 campos correctamente en los clientes para el timbrado fiscal:
    1. Nombre: Colocar la razon social como aparece en la constancia de situación fiscal (Para pruebas cualquier nombre)
    2. Código postal: Colocar el dato que aparece en la constancia de situación fiscal (Para pruebas mismo código postal que en la compañía)
    3. RFC: Colocar el RFC como aparece en la constancia de situación fiscal (Para pruebas usar XAXX010101000)
    4. Uso de CFDI: Colocar el uso solicitado por el cliente (Para pruebas colocar “Sin efectos fiscales”)
    5. Régimen fiscal: Colocar el régimen fiscal como aparece en la constancia de situación fiscal (Para pruebas "Sin obligaciones fiscales")


  5. Productos
    6. Es necesario configurar 2 campos correctamente en los clientes para el timbrado fiscal:
    1. Unidad SAT: Colocar una unidad válida de los catálogos de SAT (Para pruebas puede utilizar por ejemplo Pieza).
    2. Clave producto: Colocar una clave válida de los catálogos de SAT (Para pruebas puede utilizar por ejemplo 10101807).
    Campos opcionales fisales:
    1. Impuestos: Se puede dejar vacío y el sistema lo agrega automáticamente, si se coloca manualemnte, siempre pondrá dicho valor.
    2. Partes: Nodo adicional del CFDI para agregar partes a un producto.


  6. Diarios
    7. Opcionalemnte se pueden configurar los diarios de tipo "Facturas de clientes" para seleccionar un código postal y horario distinto al de la compañía, por ejemplo para sucursales.

    En la pestaña "CFDI" se puede colocar un código postal y horario particular para cada diario de factura de cliente.

Timbrado de facturas

Seleccionar en el documento de factura un cliente y producto configurados con su información fiscal.


Ingresar los valores fiscales en la pestaña "CFDI" como se requiera timbrar la factura.
Para timbrar el documento dar clic en el botón "Generar CFDI".


Una vez que se haya timbrado el documento, va a aparecer el fólio fiscal (UUID).


Timbrado de pagos

En una factura confirmada y timbrada, se le da clic en el botón "Registrar pago".


En la ventana emergente se debe dar clic en la opción "Validar y generar REP"


En el documento de pago se debe llenar la "forma de pago".
Cuando se requiera timbrar se le da clic en el botón "Generar REP".


Una vez que se haya timbrado el documento, va a aparecer el fólio fiscal (UUID).


Es posible incluir varias facturas en un mismo pago, una forma es seleccionando las facturas y dando clic en la acción "Registrar pago".
